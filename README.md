# 🎡 Roue FAQ Paie & RH — CNPR Centre-Val de Loire

Jeu interactif basé sur la FAQ Paie & RH du CNPR.
Hébergé via GitHub Pages.

## 🚀 Mise en ligne
1. Aller dans **Settings**
2. Section **Pages**
3. Build & Deployment → *Deploy from branch*
4. Branch : `main`
5. Folder : `/root`
6. Save

Votre roue sera disponible à une URL du type :
https://votre-nom.github.io/cnpr-roue-faq/

## 🎯 Fonctionnalités
- 10 questions issues de la FAQ CNPR
- Tirage aléatoire
- Animation fluide
- Jouable à 3–4 participants
- Compatible navigateur, SharePoint, Notion, intranet URSSAF
